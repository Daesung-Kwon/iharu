# 주아체 폰트 설치 가이드

## 폰트 다운로드

1. 배달의민족 공식 폰트 페이지에서 주아체를 다운로드하세요:
   - https://www.woowahan.com/#/fonts (배달의민족 폰트 다운로드 페이지)

2. 다운로드한 ZIP 파일을 압축 해제합니다.

3. 이 폴더(`assets/fonts/`)에 다음 파일들을 복사하세요:
   - `BMJUA_ttf.ttf` (또는 `BMJUA.otf`)
   - 또는 주아체 폰트 파일 (보통 `BMJUA`로 시작하는 파일명)

## 폰트 파일 확인

폰트 파일이 제대로 설치되었는지 확인:
```bash
ls -la assets/fonts/
```

다음과 같은 파일이 있어야 합니다:
- `BMJUA_ttf.ttf` 또는 `BMJUA.otf`

## 주의사항

- 주아체는 배달의민족에서 제공하는 폰트로, 개인 및 기업 사용자 모두 무료로 사용 가능합니다.
- 상업적 사용도 가능하지만, 폰트 파일 자체를 유료로 판매하는 것은 금지되어 있습니다.
- 가능하면 폰트 사용 시 출처를 표기하는 것을 권장합니다.
