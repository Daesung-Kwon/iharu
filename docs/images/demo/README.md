# 데모 화면 이미지

이 폴더에는 i하루 앱의 데모 화면 스크린샷이 저장됩니다.

## 이미지 파일 목록

다음 이미지 파일들을 이 폴더에 추가하세요:

1. **today-screen.png** - 오늘의 일정 화면 (활동 완료 축하 모달 포함)
2. **schedule-creation.png** - 일정 만들기 화면 (드래그 앤 드롭 인터페이스)
3. **activity-management.png** - 활동 관리 화면 (활동 목록 그리드)
4. **add-activity.png** - 새 활동 추가 모달 화면
5. **settings.png** - 설정 화면

## 이미지 규격

- **권장 해상도**: 1024x768px 이상 (태블릿 화면 비율)
- **파일 형식**: PNG (투명 배경 권장) 또는 JPG
- **최적화**: 파일 크기는 500KB 이하 권장

## 사용 방법

이미지를 추가한 후, README.md에서 다음과 같이 참조합니다:
